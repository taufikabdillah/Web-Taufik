Berkenalan Dengan Pemrograman Web

Web Programming atau Pemrograman Web merupakan istilah yang erat kaitannya dengan internet dan website. Memang benar, karena pemrograman web merupakan suatu proses pembuatan website untuk keperluan internet. Orang banyak mengenal web dengan istilah WWW atau world Wide Web.


World Wide web adalah halaman-halaman website yang terkoneksi atau dengan lainnya atau disebut juga dengan hperlink yang membentuk samudera informasi yang berjalan dengan protokol Hyperlink Transfer Protocol (HTTP)

Kelebihan Aplikasi Berbasis Web:

  1. Mudah diakses dan dapat diakses berbagai jenis perangkat seperti, perangkat mobile/smartphone, perangkat deskop, dan perangkat table.
  2. Dapat dijalankan diberbagai macam sistem operasi (OS) seperti Mac, Windows, Linux, dll
  3. Dapat diakses dari mana saja dan kapan saja, asalkan terhubung dengan jaringan internet maupun jaringan local tempat Aplikasi Berbasis Web itu berada

Kekurangan Aplikasi Berbasis Web

  • Memerlikan koneksi internet yang stabil agar tetap dapat menggunakan aplikasinya secara nyaman.
  • Memerlukan sistem keamanan yang baik, baik dari sisi server, website, koneksi yang digunakan, dll.
